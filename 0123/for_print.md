### 1.1 세로로 풀기

for i in range(2,10):
    print('-'*7 + f" [{i}단] " +'-'*7)
    for j in range(1,10):
        print(f'{i} X {j} = {i*j}')

### 1.2 가로로 풀기

`print()` 내의 `end` 인자를 잘 이용하면 쉽게 풀 수 있다.


```python
for i in range(2,10):
    print('-'*7 + f" [{i}단] " +'-'*7, end = '\t')
print('\n')
for i in range(1,10):
    for j in range(2,10):
        print(f'{j} X {i} = {i*j}', end='\t\t')
    print('\n')
```
